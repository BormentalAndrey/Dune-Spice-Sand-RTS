# Changelog

All notable changes to Dune: Spice & Sand will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of Dune: Spice & Sand
- 3 playable factions: House Atreides, House Harkonnen, Fremen
- 18 campaign missions following Dune, Book I
- Full RTS mechanics with book-accurate systems
- Sandworm AI with vibration attraction (Dune, Book I, Chapter 3)
- Shield mechanics with lasgun explosion (Dune, Book I, Chapter 5)
- Fremen worm riding (Dune, Book I, Chapter 35)
- Atreides prescience system (Dune Messiah)
- Bene Gesserit Voice ability (Dune, Book I, Chapter 1)
- Mobile-optimized touch controls
- Android build support (API 26+)
- Save system with cloud support
- Google Play Games integration
- Full audio system with Dune-inspired music
- Procedural terrain generation for Arrakis
